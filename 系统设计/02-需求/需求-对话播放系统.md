# 需求：对话播放（DialogSystem）

| 字段 | 值 |
|------|-----|
| 模块键 | DialogSystem |
| 状态 | agreed（与当前实现一致） |

## 1. 目标

按 `DialogNodeConfig` **线性播放**对话：自动逐条出消息 → 遇选项暂停 → 玩家选择后继续；与会话记忆结合，避免重复延迟与重复选项；与 `ChatUIManager`、`ChatHistoryStorageService` 集成。

## 2. 功能需求

| ID | 描述 | 优先级 | 验收要点 |
|----|------|--------|----------|
| REQ-DLG-001 | `DialogPlaybackController` 可配置各段间隔、进入会话时是否立即刷出 | P0 | 节奏符合策划表 |
| REQ-DLG-002 | 静态字典记录已见消息与已选选项节点，跨会话键隔离 | P0 | 重进对话表现符合设计 |
| REQ-DLG-003 | 玩家选项可回填玩家气泡文案 | P1 | 重进会话 UI 与记忆一致 |
