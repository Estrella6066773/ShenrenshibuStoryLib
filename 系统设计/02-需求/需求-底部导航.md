# 需求：底部导航（UI_Navigation）

| 字段 | 值 |
|------|-----|
| 模块键 | UI_Navigation |
| 状态 | agreed（与当前实现一致） |

## 1. 目标

提供 **Win 入口展开/收起**、快捷存档、存档管理、设置、关机等子页面切换；支持子物体自动解析与场景加载（见代码）。

## 2. 功能需求

| ID | 描述 | 优先级 | 验收要点 |
|----|------|--------|----------|
| REQ-NAV-001 | 开局可配置为仅显示 Win 图标，点击后展开四主按钮 | P0 | 与 UI 预制一致 |
| REQ-NAV-002 | 关机区域展开退出游戏、返回主菜单 | P0 | 二次确认或行为以策划为准 |
| REQ-NAV-003 | 未绑定页面根时按钮为占位逻辑不抛异常 | P1 | 开发中期场景可跑 |
