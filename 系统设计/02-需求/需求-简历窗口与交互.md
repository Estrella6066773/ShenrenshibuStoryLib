# 需求：简历窗口与交互（Resume）

| 字段 | 值 |
|------|-----|
| 模块键 | Resume |
| 状态 | agreed（与当前实现一致） |

## 1. 目标

提供简历列表/详情视图、拖拽与排序、选中高亮、插卡/扫描等表现桥接；与员工卡工厂、区域提供者等协作，支撑「人事管理」核心 UI 体验。

## 2. 功能需求

| ID | 描述 | 优先级 | 验收要点 |
|----|------|--------|----------|
| REQ-RSM-001 | `ResumeController` 等协调数据与生成，不破坏 Employee 模型边界 | P0 | 列表与详情数据一致 |
| REQ-RSM-002 | 支持拖拽重排、窗口区域判定（`ResumeWindowArea` 等） | P0 | 非法落点有兜底 |
| REQ-RSM-003 | 详情页动效（打字机、自上而下揭示等）可开关或配置 | P2 | 性能可接受 |
