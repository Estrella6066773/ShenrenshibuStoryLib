# 需求：压力系统（Stress）

| 字段 | 值 |
|------|-----|
| 模块键 | Stress |
| 状态 | agreed（与当前实现一致） |

## 1. 目标

提供玩家压力**统一变更入口**、**分段（Tier）**变化通知、高压/临界时**压力事件**触发与去重；可对委托施加 **assignment 范围内临时效果**；通过桥接影响员工状态与邮件等叙事。

## 2. 功能需求

| ID | 描述 | 优先级 | 验收要点 |
|----|------|--------|----------|
| REQ-STR-001 | `ApplyStressDelta` 为统一变更入口 | P0 | 所有增量路径最终经此聚合 |
| REQ-STR-002 | 分段变化对外可订阅（事件或回调） | P0 | UI/其它系统可响应 |
| REQ-STR-003 | 压力事件去重键避免同一帧/短期内重复轰炸 | P0 | 高频增量下事件数可控 |
| REQ-STR-004 | 任务 Scoped 效果带 `assignmentId` 生命周期 | P1 | 任务结束或过期后效果消失 |
