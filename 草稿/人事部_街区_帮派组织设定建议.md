# 人事部 / 当地街区 / 帮派组织：设定建议（简版可落地）

目标：细化“人事部（万事屋）”、当地街区生态与灰色势力，便于持续产出委托与冲突事件。以下为**建议稿**，保持条目化，便于你后续拆分为独立词条。

---

## 1. 人事部（社区事务与临时人力调度部 / 万事屋）

### 1.1 组织定位（建议）

- **公司对外的“问题吸收器”**：把无法标准化、风险高、需要人类承担责任的事务统一丢进人事部。
- **对内的“临时项目制”**：不按传统科层分工，而按委托快速组队、快速交付、快速结算。

### 1.2 内部结构（建议：轻量但可写剧情）

- **调度核心（玩家：HR-09）**：审核、分配、风控、绩效与惩戒/奖励。
- **现场执行组（人类/外包）**：跑腿、谈判、护送、讨债、调停、抓捕等。
- **法务接口（最小化配置）**：只负责“把事情包装成可归档的合规文本”（例如：纠纷调解、安保外包、风险处置）。
- **舆情接口（最小化配置）**：只负责“压热度/转移矛盾/定向放风”与媒体对接口径。
- **资产与装备库（可选）**：低配装备、伪装身份、一次性终端、车辆与无人机借调。

> 关键点：**不必写成大部门**，反而强调“接口化、外包化、项目制”更符合企业效率逻辑。

### 1.3 核心规则（建议：用于玩法约束）

- **可否受理**：钱到位 + 风险等级在可控阈值内（阈值由公司 KPI/审计动态调整）。
- **优先级**：公司资产安全 > 舆情 > 合同履约 > 员工生命（黑色幽默保留空间）。
- **背锅机制**：所有“不可公开流程”必须绑定到某个**人类签字主体**（外包/临时工/缓刑人员）。
- **失败处置**：默认“止损优先”，必要时切割承包方/灭口式封口（可作为道德困境事件）。

### 1.4 人事部常见委托模板（建议：可直接用）

- **纠纷类**：住户冲突、物业争执、债务纠纷、劳务纠纷（含合同奴隶）。
- **灰法类**：灰区调解、证人保护（实际是封口/换身份）、“失踪人口”找回（实际是债务追踪）。
- **黑市类**：走私渠道干预、黑客抓捕、数据回收、赃物赎回。
- **舆情类**：压热度、引导叙事、制造替罪羊、栽赃与澄清的双向业务。
- **安保类**：护送、临时清场、骚乱“降温”、社区维稳。

## 2. 当地街区（建议：做成“企业辖区”内的分层地图）

### 2.1 街区分层（建议：3 层足够）

- **A 层：明亮层（企业橱窗区）**
  - 特征：监控密度高、执法无人机常态巡逻、消费与医疗资源充足、对外宣传用。
  - 叙事用途：假象的秩序、体面人的肮脏交易、舆论中心。

- **B 层：缝隙层（居住与灰业混合区）**
  - 特征：普通住户/小商贩/外包工聚集，规则弹性大，能“谈”也能“打”。
  - 叙事用途：最适合发生委托与冲突，帮派渗透、社区纠纷、讨债、调停。

- **C 层：暗面层（维护通道/废弃区/黑市网络）**
  - 特征：基础设施老化、系统覆盖断层、黑市与非法实验的温床。
  - 叙事用途：帮派总部、反抗军据点、被公司隐藏的事故遗留地。

### 2.2 关键公共空间（建议：便于挂任务）

- **“合同大厅”**：集中办理劳务、债务、缓刑劳动签约；适合做“签字背锅”剧情。
- **“维修井口/通风塔”**：连接暗面层；适合追踪、伏击、藏匿证人。
- **“数据当铺”**：用隐私/记忆/账号换钱；适合黑客、舆情、勒索线。
- **“无人诊所”**：自动化医疗点；适合人体改造、伪造病历、药品走私。
- **“看板广场”**：巨型屏幕投放企业口号与 KPI；适合舆情事件与群体骚乱。

## 3. 组织建议：狼群帮（可用） / 鬣狗帮（可用） / 花豹帮（原住民帮派，可用） / 反抗军（可用）

> 组织名可替换；关键在于“**资源来源**、**行动风格**、**与企业的交易方式**”三要素区分开。

### 3.1 狼群帮（建议：纪律型掠食者）

- **定位**：半军事化街头势力，主打“团队协同、地盘秩序、收费保护”。
- **地盘**：B 层缝隙区边缘，控制交通节点/仓库/夜市通道。
- **收入**：保护费、运输抽成、安保外包、债务催收承包。
- **技术/武装**：中等（有改造兵、低配无人机、军规通讯）。
- **行事风格**：讲规矩、重队形、强调“我们提供秩序”。
- **与人事部关系（建议）**：
  - 可合作：人事部外包“护送/清场/调停”给狼群帮。
  - 可冲突：狼群帮反对人事部扶持其他势力，或拒绝背锅条款。
- **可用钩子**：
  - “保护费”被伪装成“物业服务费”引发住户冲突（人事部调停）。
  - 狼群帮内部出现叛徒把公司委托外泄（人事部抓捕）。
  - 狼群帮要求人事部提供“合法外包合同”洗白地盘收益。

### 3.2 鬣狗帮（建议：机会型食腐者）

- **定位**：松散联盟，擅长趁乱掠夺、倒卖、勒索；不讲规矩但极会钻空子。
- **地盘**：C 层暗面层入口与废弃区，掌握黑市渠道与走私线。
- **收入**：赃物倒卖、数据勒索、走私抽成、绑架与“赎回服务”。
- **技术/武装**：两极化（少数技术核心很强，大多数是消耗品）。
- **行事风格**：散、快、脏；爱把事做绝，擅长伪装成“事故/意外”。
- **与人事部关系（建议）**：
  - 可合作：人事部用鬣狗帮找“消失的人/货/数据”，但风险高。
  - 可冲突：鬣狗帮更倾向把人事部项目当成“可勒索对象”。
- **可用钩子**：
  - 数据当铺被鬣狗帮控制，街区居民账号被集体勒索（人事部处理）。
  - 鬣狗帮绑走证人，向公司与反抗军两头要价（人事部救援/灭口两难）。
  - 鬣狗帮散播“企业阴谋论”换流量，导致看板广场骚乱（舆情委托）。

### 3.3 花豹帮（建议：原住民帮派 / 社区守门人）

- **定位**：街区“原住民”形成的守护型势力，表面是社区互助与自保网络，必要时会以帮派方式动用暴力。
- **地盘**：B 层核心居住片区与老旧公共设施周边（学校旧址、维修站、老市场等），并掌握通往 C 层的少数“安全通道”。
- **收入**：社区互助基金、摊位与物流小抽成、情报与带路费；原则上**少做掠夺**，更像“收管理费”维持秩序。
- **技术/武装**：偏低到中等（熟悉地形、改装设备、手工陷阱、少量退伍者）；强项是“人脉/地理/情报”。
- **行事风格**：护短、排外、重名誉；对外来势力（公司外包、鬣狗帮、反抗军渗透）高度警惕。
- **与人事部关系（建议）**：
  - 可合作：人事部要进社区/暗面层办事时，需要花豹帮带路、协调住户、提供掩护口径。
  - 可冲突：花豹帮拒绝“背锅合同”，或认为人事部在替公司掩盖伤害社区的真相。
  - 可利用：公司可能尝试把花豹帮“社区自管”包装成示范项目（A 层宣传），引发内部撕裂。
- **可用钩子**：
  - 花豹帮扣押一名人事部外包队员，指控其在社区制造“事故”；人事部需要谈判或交换筹码。
  - 花豹帮与狼群帮争夺“维稳外包合同”，社区居民站队分裂（调停委托）。
  - 花豹帮发现反抗军在社区互助点招募未成年人，要求人事部“把人带走”，但公司口径要求低调。

### 3.4 反抗军（建议：叙事核心势力，但保持灰度）

- **定位**：反企业统治的地下组织，但内部派系复杂，不必写成纯正义。
- **据点**：C 层暗面层、维护通道、旧避难所；也可能在 B 层有“社区互助点”伪装。
- **资源**：偷来的设备、黑客支持、社区支持、少量退伍兵与改造人。
- **目标（建议三选一或并存）**：
  - **反 AI 管控**：拆监控、破权限、让街区“失明”。
  - **反 合同奴役**：破坏合同大厅、解救缓刑劳工/负债者。
  - **反 信息操控**：攻击看板广场、曝光企业黑箱。
- **与人事部关系（建议：多层）**：
  - 公开敌对：把人事部视为企业“脏手套”。
  - 私下交易：用情报/人质交换“放人/撤案/销毁档案”。
  - 渗透可能：反抗军安排“简历投递者”混入人事部外包队。
- **可用钩子**：
  - 反抗军劫走合同大厅的档案服务器（人事部追回/销毁/替换）。
  - 反抗军准备在看板广场发动“信息爆破”，人事部接到危机公关委托。
  - 反抗军内部派系争斗：激进派要炸基础设施，温和派要救人（人事部被迫选边）。

## 4. 组织之间的关系网（建议：一句话就够）

- **狼群帮**：要“秩序与地盘合法化”，更容易与人事部形成长期外包关系。
- **鬣狗帮**：要“混乱与利润最大化”，更像街区层面的不稳定变量。
- **花豹帮**：要“社区自保与延续”，在地理与人脉上有优势，是进入街区深处的关键门槛。
- **反抗军**：要“改变规则”，既是主线推动器也是道德困境来源。
- **人事部**：要“完成委托并止损”，既可与任何势力交易，也随时成为背锅对象。

## 5. 快速扩写用的“委托种子”（建议：每条一章/一事件）

- **债务追讨**：欠款人躲进反抗军互助点；追讨=破坏社区支持，放弃=KPI惩罚。
- **证人保护**：证人掌握企业非法证据；保护他=反抗军要人、鬣狗帮要钱、公司要封口。
- **帮派调停**：狼群帮要驱逐鬣狗帮；公司担心影响 A 层形象，要求“静默处理”。
- **舆情处置**：看板广场出现“AI不需要人类”的投放错误（或故意）；人事部要找出是谁改写了投放策略。
- **黑客抓捕**：抓到的黑客简历异常优秀，被高层强行录用进人事部项目组（冲突事件）。
