# 实现：核心消息与启动门控（CoreMessaging）

| 字段 | 值 |
|------|-----|
| 模块键 | CoreMessaging |
| 主代码路径 | `Assets/01_Scripts/Core/Messaging/` |

## 1. 职责边界

- **负责**：安装全局 `IEventBus`、静态访问点 `RuntimeMessageHub`、运行时初始化完成消息类型定义。
- **不负责**：业务命令处理（在 Assignment 等模块）。

## 2. 入口与生命周期

- `EventBusBootstrap`（见同目录）：`Awake` 中创建总线并调用 `RuntimeMessageHub.Install`。
- 上游在适当时机发布 `RuntimeInitializationCompletedEvent`（字段含 `gateKey`，默认语义 `all_systems_ready`）。

## 3. 核心类型

| 类型 | 文件 | 说明 |
|------|------|------|
| `RuntimeMessageHub` | `RuntimeMessageHub.cs` | `Bus`、`IsReady` |
| `RuntimeInitializationCompletedEvent` | 同上 | 门控载荷 |
| `EventBus` / `IEventBus` | `EventBus.cs`、`IEventBus.cs` | 订阅与发布 |
| `EventBusBootstrap` | `EventBusBootstrap.cs` | 场景安装点 |

## 4. 集成

- `AssignmentManager` 订阅门控后再订阅 `CreateAssignmentCommandEvent` 等（见 `实现-委托系统.md`）。
