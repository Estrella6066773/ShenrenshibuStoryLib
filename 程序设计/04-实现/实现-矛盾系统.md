# 实现：矛盾系统（Conflict）

| 字段 | 值 |
|------|-----|
| 模块键 | Conflict |
| 主代码路径 | `Assets/01_Scripts/ConflictSystem/` |

## 1. 职责边界

- **负责**：任务域矛盾累积、阈值触发、模板筛选、玩家协调 `TryResolveConflict`、结算幂等、历史与活跃案件、存档。
- **不负责**：邮件 UI 渲染（MessageSystem）；压力值计算（Stress）。

## 2. 分层结构

| 层 | 路径 | 说明 |
|----|------|------|
| Application | `Application/ConflictSystemService.cs` | 编排入口 |
| Domain | `Domain/*` | 状态机、运行时模型、键工厂 |
| Config | `Config/*` | SO 模板与分类 |
| Integration | `Integration/*` | 经济/压力/委托网关、事件总线接口 |
| Save | `Save/*` | 序列化映射 |

## 3. 场景入口

- `ConflictSystemBehaviour`：挂载服务、连接 `PlayerEconomySystem` 与总线等（见场景配置）。

## 4. 桥接

- `ConflictEmployeeStateBridge`：员工 `InConflictEvent` 与任务槽位 `workerId` 对齐。
- `ConflictMailBridge`：叙事投递。
