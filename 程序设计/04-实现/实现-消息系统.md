# 实现：消息系统（MessageSystem）

| 字段 | 值 |
|------|-----|
| 模块键 | MessageSystem |
| 主代码路径 | `Assets/01_Scripts/MessageSystem/` |

## 1. 职责边界

- **负责**：邮件/聊天/日志 UI 管理器、聊天视图数据、历史与频道存档、邮件派发与收件运行时、SO 配置（邮件事件、频道、角色、剧情等）。
- **不负责**：对话节点图定义本体（`DialogSystem/Data` 与播放控制器协同）。

## 2. UI 入口

| 类型 | 路径 |
|------|------|
| `MessageUIManager` | `message/MessageUIManager.cs` |
| `MessageTabUIManager` | `message/MessageTabUIManager.cs` |
| `ChatUIManager` | `message/ChatUIManager.cs` |
| `MailUIManager` | `message/MailUIManager.cs` |
| `LogUIManager` | `message/LogUIManager.cs` |

## 3. 模型与服务

| 类型 | 路径 |
|------|------|
| `ChatHistoryStorageService` | `Model/ChatHistoryStorageService.cs` |
| `ChatChannelArchiveService` / Models | `Model/ChatChannelArchive*.cs` |
| `MailDispatchService` / `MailRuntimeInbox` | `Model/MailDispatchService.cs` 等 |
| `ChatStorySO` 等 | `Model/ScriptableObjects/` |

## 4. 剧情运行时

- `Runtime/ChatStoryPlayer.cs`、`ChatStoryChatBridge.cs`：剧情驱动与聊天桥接。

## 5. 编辑器

- `Assets/01_Scripts/Editor/ChatStorySOEditor.cs`、`MailEventDatabaseSOEditor.cs` 等。
