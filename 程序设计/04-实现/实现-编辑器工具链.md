## 1. 脚本清单

| 文件 | 职责摘要 |

| `ShenRenShiBuAssetMenu.cs` | 菜单「神人事部/创建/*」：标签库、标签、简历模板、分支树、频道注册表、角色库、剧情、邮件数据库等；内含 `BasePath = Assets/00_Data/` 与 `04_Data` 下对话/邮件路径常量 |
| `AssignmentSystemSetupMenu.cs` | 「神人事部/场景/搭建委托系统」：生成委托根、模板组、运行时节点，挂 `AssignmentControllerBehaviour` 等 |

| `CatQuestAssignmentTemplateMenu.cs` | 示例委托模板相关菜单 |
| `AssignmentTemplateSOEditor.cs` | 委托模板 Inspector 扩展 |
| `ChatStorySOEditor.cs` | 剧情 SO 编辑器 |
| `MailEventDatabaseSOEditor.cs` | 邮件库编辑器 |
