# 实现：数据资产布局（DataAssets）

| 字段 | 值 |
|------|-----|
| 模块键 | DataAssets |
| 主资源路径 | `Assets/04_Data/`（及工具中 `Assets/00_Data/` 创建目标，见下） |

## 1. 运行时内容目录（仓库现状摘要）

| 区域 | 路径模式 | 用途 |
|------|-----------|------|
| 对话与剧情 | `04_Data/对话dat/频道配置/`、`角色库/`、`剧情演出/` | `ChatChannelRegistry*`、`ChatRoleDatabase*`、`ChatStory*` 等 |
| 消息系统 | `04_Data/MessageSystem/` 等 | 邮件相关数据库（以 `ShenRenShiBuAssetMenu` 常量为准） |
| 标签 | `04_Data/标签系统/标签类型/*` | 多子类（性格、义体、状态…）`EmployeeTagSO` |

## 2. 编辑器创建路径（与代码常量对齐）

`ShenRenShiBuAssetMenu` 中：

- 部分资产默认写入 **`Assets/00_Data/`** 子目录（`Tags/`、`Templates/`、`Databases/`、`Assignments/`、`Portrait/`）。
- 对话与邮件创建入口指向 **`Assets/04_Data/`** 下子路径。

**工程注意**：若仓库以 `04_Data` 为唯一事实源，建议在计划中统一「新建资产默认路径」，或改菜单常量与现网数据一致。

## 3. 关联模块

- Employee / MessageSystem / Assignment 的 IMPL 中 SO 类型均依赖此处资产。
