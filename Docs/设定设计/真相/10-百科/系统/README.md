# 人工智能系统（桥）

本目录仅收录**管理者人工智能实例**及其谱系、职能与事变因果——即世界观中的「**桥**」（资产代号如 **B-9002**），而非人事部的办事制度。

| 类型 | 说明 |
|------|------|
| **是什么** | **桥梁级管理型人工智能**（*Bridge-grade Administrative AI*），可独立履职；字母 **B** 代表 **Bridge（桥梁）**，对外类型常登记为「桥」 |
| **不是什么** | 外包审批、委托优先级、员工冲突处理等**制度规则**（详见 [../规则/README.md](../规则/README.md)） |
| **勿混用** | 叙事中若以「水电不能断」等人物口语指代供水供电及清运事务，不在本目录收录范围 |
