# 对人类必要性的工具化界定

## 第一条　本条旨趣

本条只从**制度层面**收紧「人类在自动化秩序中为何仍被保留」的答案，防止叙事滑向「科技进步则人的价值自然抬升」一类误读。

## 第二条　人类在制度中的功能定位

人类在当下体系中承担两项核心功能：一是**承接具体责任**，使制度决策可追溯到可辨识的人类主体；二是**监督系统运行**，在执行过程中发现并反馈系统的漏洞与偏差。短期内尚不便整套交给机器，但这**并不等于**人在权利与尊严上会被平等对待。

## 第三条　与伦理叙事的关系

上述界定是批判性叙事的**前提**，不是价值观宣言。若要写「人性化改革」，必须交代背后的真实驱动力——利益交换、恐惧、反抗压力、丑闻等——不能预设「善意凭空降临」。
