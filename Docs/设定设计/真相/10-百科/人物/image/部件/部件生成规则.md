# 胸像部件生成规则

> 部件对照 `image/参考/` **手绘或逐像素整理**。

## 一、用途与范围

- **用途**：模块化胸像（头像）拼合；通用部件库对照 `image/参考/` 建立，供多名 NPC / 干部立绘复用。
- **范围**：仅 **4:5 胸像**（**宽:高 = 4:5**，逻辑 **96×120 px**，肩以上取景），对齐 `image/参考/` 的像素尺度、色调与气质。
- **色调真值**：**仅**以**未救援**参考（无底部 `RESCUED` 条）为准；带「已救援」条的图整体被**压暗**，只可借构图与形状，**不可**吸色、不可当色板样本。
- **禁止**：任何游戏 UI（含底部 `RESCUED` 青条、`+` 图标、救援字样）；参考图上的该条仅作来源说明，**不得**画入部件或成品立绘。

## 二、画面规格

| 项目 | 要求 |
|------|------|
| 比例 | **宽:高 = 4:5**（先写宽、后写高；例 96 宽 × 120 高） |
| 画幅 | **96×120 px** 胸像 |
| 参考预处理 | 原图若含底部 `RESCUED` 条，**先裁掉 UI**，再套进 **96×120** 取景 |
| 工作放大 | Aseprite 等用 **4×** 整数倍画布（**384×480**），导出 **最近邻** 缩回 **96×120** |
| 背景 | 作画、对照、拼合预览：**绿幕**（`#00FF00`）；**部件 PNG 导出仍为透明** |
| 像素风格 | 硬边像素、无抗锯齿；**不存在半透明内容**（仅 **0% / 100%** 不透明度，禁止抗锯齿边、羽化、图层透明度低于 100%） |
| 填色 | 每区域 **2–3 档实色**（高光 / 主色 / 暗部），过渡用邻接色块，**不用**发丝级细节 |

### 造型语言（对齐 `image/参考/`）

与十张参考一致：**以色块实色为主**，辅以**少量结构线**；**不是**线稿起稿再上色，也**不是**色块与线条对半的线描风。

| 层级 | 要求 |
|------|------|
| **色块（主体）** | 脸、发、衣用大面 **2–3 档实色** 分区；明暗靠**邻接色块**衔接，头发**分块填色**，不勾发丝线 |
| **结构线（辅助）** | 仅用于局部：**嘴缝**、**上下睑**、**眉**、**镜框**、**鼻梁影** 等，多为 **1 px** 实色线 |
| **外轮廓** | 靠深灰发、深衣、肤色与背景（绿幕）的**色块对比**成形；**禁止**人物外圈纯黑描边一圈 |

**禁止**：线稿主导画面、整圈硬描边、发丝级勾线、把「由色块和线条组成」理解成线条与色块同等权重。

## 三、部件清单与表格式样

共 **11 类**，**一类一表**（一张 PNG 横向排列多枚变体，或一格一文件，团队自定；下文统称「素材表」）。

**表内顺序**（从左到右建议与参考图文件名排序一致，便于对照）：

1. 脸型  
2. 脸部装饰物  
3. 前发  
4. 后发  
5. 眼部  
6. 眼部饰物  
7. 嘴  
8. 鼻  
9. 眉  
10. 耳  
11. 衣物  

**表内禁忌**：一张表里只能出现**上述一类**；不得混画脸型与头发、不得带 UI。

**文件建议路径**：

- 通用变体：`image/部件/素材表-{类别}.png`  
- 若某角色不走通用库、需单独一套：另建 `image/部件/{角色ID}/`（以该角色设定为准，**不**写入本规则）

## 四、叠放顺序（从下到上）

拼合时按此顺序粘贴，避免刘海压住眼镜、衣领压住下巴等穿帮：

1. 后发  
2. 脸型  
3. 脸部装饰物  
4. 耳  
5. 鼻  
6. 嘴  
7. 眼部  
8. 眉  
9. 眼部饰物  
10. 前发  
11. 衣物  

## 五、对齐锚点

所有部件应共用同一套锚点，剪贴时不漂移：

- **水平**：两眼连线中点，或单眼可见时眉心—鼻尖中轴  
- **垂直**：下巴最底点（与衣领上缘衔接）  
- 单格内部件应**居中留空**，格与格之间留 **2–4 px** 透明缝，便于批量裁剪  

## 六、色板与光影（对齐 `image/参考/`）

### 未救援 / 已救援（色调必读）

十张参考里，底部带 `RESCUED` 的为**已救援**图（游戏状态叠层后人物整体偏暗）；**无**该条的为**未救援**图（肤色、发色、衣服明度正常）。

| 状态 | 文件 | 用途 |
|------|------|------|
| **未救援**（色调真值） | `…2127…`、`…2132…`、`…2133…` | 吸色、定档、部件库与成品立绘的**唯一**色板来源 |
| **已救援**（仅形状） | `…2124…`、`…2125…`、`…2126…`、`…2128…`、`…2129…`、`…2130…`、`…2131…` | 可对照刘海、眼型、耳位、雀斑位置等；取色后须**提亮并改回**未救援档，禁止直接沿用其深色 |

已救援图裁掉 UI 后仍偏暗，取色须改回未救援档。

### 色板

整体 **低饱和、偏灰**，类似证件照 / 视觉小说胸像，**不要**鲜艳色。明度以**未救援**三张为准：肤色偏亮灰米、发与衣对比清晰。画布与预览底为**绿幕**（`#00FF00`），勿在部件层画绿底。

| 区域 | 色相倾向 | 档位数 | 说明 |
|------|----------|--------|------|
| 皮肤 | 灰米、灰粉 beige | 2 | 下颌与眼眶下缘可加一档暗部；避免纯白高光 |
| 头发 | 灰金、棕灰、近黑 | 2–3 | **分块填色**；用色块分界，不勾发丝线 |
| 眼白 / 巩膜 | 浅灰米、略亮于肤 | 1–2 | 默认档在虹膜上下或内外角留 **1–2 px** 条；**2132** 等扩展档常几乎不分巩膜 |
| 虹膜 / 瞳孔 | 深褐、深灰 **实色块** | 1–2 | 默认：**竖向块**、与瞳孔合并（2127 / 2130 / 2133）；上睑 **1–2 px** 深色线；**2124** 为半睁 + 独立 **约 4×3 px** 暗核 |
| 眉 | 同发或略深 | 1 | 多为 **1 px 高** 短横线或微弧 |
| 鼻 | 同肤或略深 1–2 px | 1 | 常仅为鼻梁影或两点鼻孔，勿写实塑形 |
| 嘴 | 暗灰线或略深肤色 | 1 | 闭合嘴多为 **3–5 px 宽** 横线 |
| 外衣 | 炭灰、近黑 | 2 | 西装、夹克、制服领 |
| 内衣 / 衬衫 | 灰白、浅灰 | 1–2 | 领口、V 领、衬衫领 |
| 雀斑 | 略深于肤的灰褐点 | 1 | 单点 **1 px**；疏朗分布，勿画成噪点墙 |
| 痣 | 深褐 / 近黑 | 1 | 单点或 **2×2 px** 实色块；位置固定可当作角色锚点 |
| 红晕 | 灰粉、暗玫瑰（低饱和） | 1–2 | 颊部 **2–6 px** **实色**块；与肤色用邻色衔接，**禁止**半透明晕染 |

**造型**：遵守第二节「造型语言」——色块为主、少量结构线；外轮廓用色块对比，**不用**纯黑描边一圈。

## 七、分类绘制要点

### 脸型

- 含脸颊、额头（露发处）、下颌、颈上段；**不含**头发、五官、衣领。  
- **不含**雀斑、痣、红晕、伤疤、纹身——上述归入「脸部装饰物」。  
- 形：椭圆至略尖下巴；颈影用暗一档肤色，**1–2 px** 即可。

### 脸部装饰物

画在**肤色之上**的面部标记与肌理，与「眼部饰物」（眼镜等）无关。一张表内可并列多种**位置方案**，但每格仍只画装饰本身（透明底），不附带完整脸型。

| 类型 | 说明 | 常见位置 |
|------|------|----------|
| **雀斑** | 深 1 档的散点，**1 px** 为主 | 鼻梁两侧、颧骨、眼下方 |
| **痣** | 实色小点，可略大 | 嘴角旁、鼻翼、颊中（按角色定） |
| **红晕** | 低饱和灰粉块，边缘可与肤色过渡 | 颧骨、眼下、耳前颊区 |

参考真值：

| 参考文件 | 说明 |
|----------|------|
| `微信图片_20260513160636_2129_1494.png` | 鼻梁/上颊雀斑位置 |

十张参考里**多数无**明显雀斑/痣/红晕；`素材表-脸部装饰物.png` 须含一格 **「无」**（空透明），再按需加 2129 雀斑、自绘痣/红晕变体。

绘制要求：

- 色数遵守第六节表格，**低饱和**；红晕不用荧光粉；**全实色、无半透明**。  
- 不画义体线、不画 UI、不画眼镜（眼镜仍在「眼部饰物」）。  
- 拼合层在「脸型」之上、「耳 / 鼻 / 嘴 / 眼」之下；前发可部分盖住颊部红晕。

### 前发

- 刘海、侧分、遮眼发帘；可不对称。  
- 遮眼发帘、侧分等按变体需要绘制，参考图里已有多种范例。

### 后发

- 耳后、颈侧、肩上方发量外轮廓；与前脸衔接处勿留透明缝（拼合时再盖在前发下或上，按发型定）。

### 眼部

对照 `image/参考/` 时，先将人物缩放到 **96×120** 再与下表核对（与第二节逻辑格网一致）。

**默认档：正常睁眼**（`素材表-眼部.png` 主行）

眼裂清楚、虹膜**竖向实色块**完整露出，巩膜条可见，带 **1 px** 高光。——**不是**半睁、睡眼、虹膜贴满无留白、或**无高光死眼**（2132）。

| 指标（96×120，单眼） | 默认睁眼（对齐参考） |
|----------------------|----------------------|
| 外形 | 杏仁形，略宽于高；外角可略低于内角（2127 最明显） |
| 可见开口 | 高约 **6–8 px**、宽约 **10–12 px**（2127 最大；2133 约 **6×10**） |
| 上睑 | **1–2 px** 深色线；下睑多为肤色/巩膜交界，**不**画第二圈黑线 |
| 巩膜 | 浅灰米色 **1–2 px** 条，包在虹膜外缘 |
| 虹膜 / 瞳孔 | **一块**深褐/深灰竖矩形（虹膜与瞳孔不分开画）；占开口高度约 **70–85%** |
| 高光 | **1 px** 浅色点：2127 多在虹膜**左上**；2130 / 2133 多在**上外侧** |
| 间距 | 两眼中心距约 **22–26 px**（约 **2 倍**单眼宽）；内眼角空约 **10–12 px** |
| 竖向位置 | 眼裂中心约在脸高 **40–48%**（眉下至下巴） |

| 参考文件 | 眼型（对照原图） | 默认档 / 扩展 |
|----------|------------------|---------------|
| `…2127…` | 圆杏仁、**开口最大**；竖块虹膜 + 左上 **1 px** 高光；巩膜宽 | **默认**（未救援，形状 + 色） |
| `…2130…` | 大杏仁、上睑线厚；竖块虹膜 + 上外 **1 px** 高光 | **默认**（已救援，仅形状） |
| `…2133…` | 中等偏窄 almond；巩膜条清楚；**1 px** 高光；**无眼镜** | **默认**（未救援，形状 + 色） |
| `…2124…` | **半睁**、开口扁；**约 4×3 px** 独立暗瞳孔核；方框镜 | **扩展**（镜在「眼部饰物」） |
| `…2125…`、`…2126…` | 半睁 / 睡眼；上睑压低，虹膜块仍大但开口扁 | 扩展 |
| `…2128…` | 单眼可见（另一侧被发遮）+ 垂睑 | 扩展 |
| `…2129…` | 半睁；上睑折线 + 虹膜块（可兼对照雀斑位置） | 扩展 |
| `…2131…` | 半睁、偏「无神」；虹膜块大、高光弱 | 扩展 |
| `…2132…` | **扁垂眼**；虹膜整块填死、**无高光**、巩膜难辨 | 扩展（可兼耳部；**不作**默认眼） |

绘制要点：

- 结构：**色块**（巩膜、虹膜）+ **1 px 结构线**（上睑）；禁止半透明、禁止渐变虹膜。  
- 刘海遮眼：另做「单眼 / 双眼 / 全遮」构图，**开口尺寸仍按默认档**（2127 口径）。  
- `素材表-眼部.png` 建议顺序：**2127 → 2130 → 2133**；2124 与 2125–2132 放入**扩展行**，不与默认混排。

### 眼部饰物

- 眼镜框、眼罩等；无则表可留空或省略文件。  
- 方框镜真值：`…2124…`；框线 **1 px** 中灰，桥连接两镜片，**不**画进 `素材表-眼部.png` 默认行。  
- **2133 无眼镜**，勿从 2124 误套镜框。

### 嘴

- 仅唇线 / 嘴缝；张开嘴另做变体时再增表。

### 鼻

- 鼻梁影、鼻头、鼻孔暗示；体量极小。

### 眉

- 独立一层，便于换表情；与刘海重叠处在拼合时以前发为准。

### 耳

十张参考里**仅下列四张**可作耳部变体真值（其余图耳被发帘挡死或几乎不可见，不宜抠图当「耳」部件）：

| 参考文件 | 可见度 | 要点 |
|----------|--------|------|
| `微信图片_20260513160638_2132_1494.png` | 双耳完整 | 短发后梳，耳露在发外；外缘略深、内侧少量阴影；高约 **12–14 px**（逻辑格网），上缘约齐眼角、下缘约齐鼻梁 |
| `微信图片_20260513160639_2133_1494.png` | 双耳（一侧更清晰） | 侧分短发；简单外轮廓 + 内侧 1–2 px 凹影；上缘约齐眉、下缘约齐鼻尖 |
| `微信图片_20260513160637_2130_1494.png` | 双耳（部分贴发） | 波浪长发；上缘约齐眼中部 |
| `微信图片_20260513160634_2126_1494.png` | 双耳 | 齐刘海；左侧略被发束挡 |

绘制与重绘要求：

- 与肤色同系，**2 档**即可（外缘略深 + 内凹阴影）；勿加发丝或发片。  
- 体量 **C 形或圆角块**，宽 **3–6 px**、高 **8–14 px**（按上表四张缩放至逻辑尺寸）。  
- 做 `素材表-耳.png` 时：建议从左到右依次为 **2132 → 2133 → 2130 → 2126**（由露耳最完整到略挡发）。  
- 拼合时耳层在「脸型」之上、「鼻/嘴」之下；后发、前发可盖住耳廓上沿。

### 衣物

- 肩、领、胸前第一截；深外套 + 浅衬衫领为常见组合。  
- **不要**画徽章、救援标、部门 UI。

## 八、从参考图提炼变体

`image/参考/` 现有十张图，建议至少覆盖：

- 发型：直刘海、中分、侧遮眼、短束、波浪肩发  
- 眼型：**正常睁眼**默认 **2127 / 2130 / 2133**；半睁、睡眼、无高光扁眼（2124–2132 等）见第七节「眼部」扩展行  
- 饰物：无镜 / 方框镜（**2124**）  
- 耳：**2132、2133、2130、2126**（见第七节「耳」表；其余参考不提供耳变体）  
- 脸部装饰物：**无**（默认）+ 雀斑（2129）+ 自绘痣 / 红晕变体  
- 衣：西装+衬衫、夹克、背心、无领背心  

每张参考只取**该类别**像素重画在透明底上，勿整图缩小当部件。

## 九、验收清单

- [ ] 比例 **宽:高 = 4:5**，逻辑尺寸 **96×120 px**  
- [ ] 最近邻缩放无糊边；参考图已裁 UI 并套 **96×120**  
- [ ] 作画 / 预览用绿幕；素材表 PNG 仅透明底、无绿幕像素  
- [ ] 全图无半透明像素（导出前检查 alpha 通道仅 0 与 255）  
- [ ] 造型为色块为主 + 少量结构线；无整圈黑描边、无线稿风、无发丝勾线  
- [ ] 每类素材表仅含一类元素  
- [ ] 无 RESCUED / 青条 / 多余 UI  
- [ ] 色板与未救援三张（2127 / 2132 / 2133）对照，未误用已救援图的偏暗色  
- [ ] 默认眼部对齐 **2127 / 2130 / 2133**；2124 仅作半睁扩展，2132 不作默认开眼  
- [ ] 叠放顺序试拼一张，下颌与衣领、刘海与眼无错位  
- [ ] 源文件保留 Aseprite / PSD 时，图层名与上表 **11 类**一致  
- [ ] 雀斑 / 痣 / 红晕仅在「脸部装饰物」层，不与脸型、眼部饰物混表
