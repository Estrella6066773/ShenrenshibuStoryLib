# 世界历史总线（TIM-03）

本文给出**世界级历史脉络**，为城市史、组织史以及宣传与话语口径的演变提供上游骨架；某一城、某一街的细部见 [TIM-02](TIM-02-城市历史母线.md) 及设定真值百科。

## 写作口径

- 结构叙述优先：每期以 **制度结构 → 变化机制 → 历史后果** 展开。
- 设定事实术语使用「**合作社**」；角色侧常见口径为「**彼岸**」（禁忌化与污名化见分期 III）。
- 与帮派相关的组织细节不在此重复创设；「狼群」的由来及其与家族战争的衔接，在城市尺度上的说明见 [ORG-GANG-WOLF](../10-百科/组织/帮派/ORG-GANG-WOLF-狼群.md)、[TIM-04](TIM-04-前任管理员执政与第九区事态背景年表.md)。

## 世界历史分期

### I. 联合州主导时期（早期稳定阶段）

- **结构**：联合州以地缘与领土关系构建合法性，维持区域秩序与宣传能力。
- **同期外部**：另一大陆的合作社体系从跨国协作起步，逐步增强对生产与组织的控制力。
- **后果**：两套治理逻辑并存，认知与制度差异持续扩大。

### II. 合作社崛起时期（生产力重组阶段）

- **结构**：合作社通过掌控生产资料与核心生产力，形成可与传统国家竞争甚至替代的治理能力。
- **治理特征**：普遍采用「人类 + AI」协同治理，AI 处于治理与决策核心位置。
- **后果**：联合州一侧的传统制度工具在面对该类竞争时逐步钝化。

### III. 认知战与禁忌词成型（约五十年前）

- **结构**：联合州将合作社系统污名化，并与「AI 暴政 / 人类工具化」叙事捆绑。
- **传播**：公共语境中「合作社」逐步被「彼岸」等替代表述置换。
- **后果**：形成长期的术语禁忌与对跨境信息的先验敌意。

### IV. 联合州失序与崩解（中后期断裂阶段）

- **结构**：联合州未能完成与之相称的社会结构改革，内部矛盾与经济问题日积月累，最终瓦解，政治版图碎裂为各州高度自治的局面。
- **对外**：合作社一侧并不一定寻求全面的武力征服，但长期施加经济压力并推行己方标准。
- **后果**：旧主权叙事空心化，区域治理进入碎片化与企业化过渡。

### V. 后联合州时期（企业辖区化阶段）

- **结构**：解体后各地出现各自为政、城市交易、州级政体企业化转型等现象。
- **城市层面（与 `[城市A]` 对齐）**：该城由殖民史路径建立；在后联合州阶段经出售转入企业辖区治理，**按部门切块、按指标考核**成为常见的治理方式（城市细节见 [TIM-02](TIM-02-城市历史母线.md)）。
- **后果**：局部秩序与效率回升的同时，合法性争议与社会张力长期存在。

### VI. 当下阶段（灰区增殖与叙事拉扯）

- **结构**：**彼岸**境内**西西里岛**发生家族政治清算（叙事里常称「家族战争」），长期牵动跨境流动与管辖边界。就 `[城市A]` 而言，此事与**此前二十五年**间第九区人事执行、外撤家族汇集为「狼群」、同**自由人联合阵线**的战事以及废墟区的形成，**在时间上如何逐年对应**，见 [TIM-04](TIM-04-前任管理员执政与第九区事态背景年表.md)；具体组织设定另见 [ORG-GANG-WOLF](../10-百科/组织/帮派/ORG-GANG-WOLF-狼群.md)、[ORG-MOV-FUF](../10-百科/组织/ORG-MOV-FUF-自由人联合阵线.md)。
- **认知**：
  - 「彼岸」既是现实里的跨境威胁标签，也是宣传长期塑造的恐惧符号；
  - 官方口径、民间流言与档案证词并行，若要还原世界层面的实情，须依靠可核对的事实链条。
- **后果**：人事外包、企业与灰区秩序服务、城市舆情之间的张力，构成常态叙事素材。

## 当前可确认的长期趋势

- **治理**：领土主张的说服力相对下降，生产力与组织效率日益成为合法性的依据。
- **权力**：单一国家的统一治理减弱，企业与多种地方主体分担治理职能。
- **认知**：单一官方叙事难以垄断舆论，多种口径并存、彼此竞争。
- **风险**：灰区组织活跃与跨境信息真伪混杂，成为难以消除的结构性不确定因素。
