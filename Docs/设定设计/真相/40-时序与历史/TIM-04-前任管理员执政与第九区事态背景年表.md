# 前任管理员执政与第九区事态背景年表（TIM-04）

## 第一条　条文性质

本条收录**前任管理员（管理者人工智能实例）**自执掌第九区并在**人事部执行体系**中担任关键职务起，至**现任桥 B-9002 启用**为止，作者侧用于对齐百科与系统的**相对时间顺序**（「执政后第 N 年」）。本条**不是**主线分幕剧本或必经剧情清单；绝对纪年与叙事当下的换算须服从 [WLD-00](../00-基石/WLD-00-世界纪年与治理总纲.md)；**具体公元年份须待通盘年表定稿后再统一填实，不得仅凭本表擅自写死。**

因果分层、信息分级与深层伏笔仍以 [SYS-PREV-BRIDGE](../10-百科/系统/SYS-PREV-BRIDGE-前任管理员与第九区废墟事变.md) 为准；本条**不能代替**该条关于「表层／中层／深层」的划分。**玩家情报层级（P0–P3）**以 [治理/02-可见性与剧透分级.md](../../治理/02-可见性与剧透分级.md) 第五条及各 `CHR-*` 第二条为准。

## 第二条　总跨度

- **管理者执政起点 → 现任启用**：共 **二十五年**（启用点为叙事「当下」接口之一）。

## 第三条　相对年表（执政后第 N 年）

专名 **无人器械** 指正由人工智能托管运行的器械（含**民用级**与**军用级**两档）；**军用级**投产见第 17 年末–第 18 年初「伊莱出逃」行，此前仅**民用级**。

| 年份（相对） | 事件要点 | 投放 | 玩家情报 |
|-------------|----------|------|----------|
| 第 2 年 | **彼岸**境内**西西里岛**爆发家族政治清算（叙事侧常称「家族战争」）；地缘用语层级见 [TIM-03](TIM-03-世界历史总线.md)、[ORG-GANG-WOLF](../10-百科/组织/帮派/ORG-GANG-WOLF-狼群.md)。 | 中层 | P2 |
| 第 3 年 | **第一批**外撤家族抵达第九区，由前任管理员主导的人事部执行体系予以收编；此后持续吸纳家族战争残党及其他投奔者，逐步形成对外统称为「狼群」的家族联合体。 | 中层 | P2 |
| 第 7–10 年 | **[自由人联合阵线](../10-百科/组织/ORG-MOV-FUF-自由人联合阵线.md)**在 `[城市A]` 起义；辖区多头治理之下，联合阵线占领部分城区（企业口径称 **沦陷区**）。前任管理员时任**企业编制内人事部最高执行管理者**，其在第九区的意志往往被视作**区政府立场**。动员 **狼群**（流亡家族谱系；战时企业对外的统战话术与狼群对外自我标榜多为「家园卫士」「护民官」，**实质仍是黑帮性质的家族联合体**）与 **防卫科**，抵御来自沦陷区方向的攻势。**彼岸志愿者**（抱理想主义动机前来支援者）在战争初期**一致支持联合阵线**。约第 **10** 年企业武装大举镇压后，第九区方面仍有余力**协助**企业武装将联合阵线**清剿出城**，并出动雇佣武装协战。联合阵线在更广战线上曾接管或攻占多座城市；企业反攻后，**残余势力**退入**联合州北部支持者较多的地区**，在其他地区仍有影响力。 | 中层 | P2 |
| 第 15 年 | 上级要求清理第九区约**四分之一**辖区土地以搭建巨构；若执行将摧毁大量关键**工业区**，等于推翻前任管理员多年治理成果。前任管理员认清上层态度；在 **双面砧**计划框架下，其使命中**偏重第九区、而非无条件服从企业**的一面逐渐显露，遂**暗中联络联合阵线与抵抗网络**，策动骚乱以阻挠计划。**摩恩夫人**因征地心灰意冷**卸任**服务科科长；卸任时**亲自**委任**下一任**科长（姓名留白），并将弟子 **艾萨克·穆尔** **调出服务科**，调往第九区**偏远地区**驻点任**精英科员**、兼**组长**（**非**上级问责；决定人：摩恩）。**瑟琳娜·昼** 已在人事部一线，受派系罪证档案束缚。前任管理员反叛意图**尚未公开**。 | 中层 | P2 |
| 谋划暴露后（约第 15 年末–第 17 年） | 企业武装压制前任管理员反叛；总部**顾及**前任管理员身上**宝贵资料**（含 **双面砧** 相关样本与 **无人器械** 产线信息），**希望能获取**，因而**一直举棋不定**，未能果断歼灭，现场**久攻不下**。此间前任管理员**仅**维持**民用级无人器械**的投产与改装；即便被观测，**亦未**引起军部按军用威胁立案。**军武部**军官 **科恩·沃洛克** 作为现场指挥之一投入第九区围堵（**非**「申请升级火力被驳回」主线）。 | 中层 | P2 |
| 第 16–17 年 | 反叛意图逐步暴露；前线仍以**民用级无人器械**为主。艾萨克在**偏远驻点**履职。摩恩所委任之服务科科长**仍在任**（约自第 15 年起）。 | 中层 | P2 |
| 第 17 年末–第 18 年初 | **军用级无人器械**在此窗口**才开始投产**（与民用改装线衔接，属前任管理员与伊莱布局一环）。前任管理员 **桥专属人类顾问** **伊莱·索伦** 按布局**假叛逃**；向军部/总部提供 **无人器械已部署、军用级已投产/正在投产** 情报，使军部**警觉**并打破「仅民用、可观望」的判断（借势社会对人工智能的恐惧，见 [ENT-AIBACKLASH](../10-百科/文化/ENT-AIBACKLASH-人工智能污名与第九区事变情绪遗产.md)）。**上交损毁母带**；私藏另一容器内**完整前任管理员副本**（公司不知，深层）。 | 深层 | P3 |
| 第 18 年 | 军部经伊莱情报**警觉**军用级威胁后，批准动用**毁灭性武器**消灭前任管理员、清除隐患。**[军武部](../10-百科/组织/赫利俄斯/ORG-MILDEPT-军用武装管理部.md)** 以**军事级攻势**摧毁旧人事部中枢及相连反抗区域，废墟区由此形成。**人事部熟悉当地情况的骨干人员突然大量死亡导致人员空缺**。摩恩所委任之服务科科长于本部**阵亡**。**艾萨克·穆尔**人在**偏远驻点**，仍遭毁灭性打击**余震**波及，**失去双腿**（后装基础义腿；**非**服务科科长）。因 **ANVIL-BIFORM** 机密，**表面上**归咎时任**军武部军官**、现场指挥 **科恩·沃洛克**「处置不力」（**非**防卫科科长）、掩盖总部**举棋不定**及**未及时按军用威胁处置**的失误；**事后**科恩自军武部**贬出**，降职留任**人事部第九区分部** **防卫科**科长（**收拾烂摊子**）。**伊莱·索伦**因情报功过**踢出双面砧计划**、**委任人事部部长**主持重建。 | 表层+中层 | P1–P2 |
| 第 18 年后（重组） | 伊莱**逐个委任四科长**：科恩已贬留第九区分部防卫科 → 艾萨克约第 19–22 年**回任**服务科 → 瑟琳娜被以第一区罪证**胁迫任职**人力科长 → 凌薇于重组后约 1–3 年内自第一区**下放**财务科。 | 中层 | P1–P2 |
| 第 18–24 年（共 7 年） | 废墟地下为**碎片化低级备份 + 写死指令的低级智能**（目标：重建可承载前任管理员的服务器）；同期 **军用级无人器械** 与产线节点藏于防空洞/地堡等，在 [废墟区](../10-百科/地点/PLC-RUINS-废墟区-第九区事变遗留片区.md) **专精游击**（伏击、诱杀、切断补给与通讯），作风**阴险狡诈**，使清剿与搜查**作战困难**。企业多次**强行探索**废墟（取样、拆机、追产线），**伤亡与装备损耗代价过于高昂**，难以持续。七年间未能稳定夺回地下工程或完整样本；**军部**经七年清剿与搜查，正式判**不值得回收** → 集团据此启动第二次实验 **B-9002**。**表层政务**由 **伊莱·索伦** 以部长身份主持重建。废墟一带 **无人器械活动** 归因不明（伊莱已被踢出计划）。伊莱**私藏完整副本**（≠废墟碎片工程，深层）。艾萨克在基建维护期或经废墟 **②** 投射至 **无人器械上的低级智能体容器**（②之分身）**间接**接触（**未**接触①完整副本或前任管理员本体全能意识，深层）。 | 分层见下 | P0 结果 / P2 判因 / P3 智能 |
| 第 25 年 | **启用现任 B-9002**；授权收窄为特别编组（授权），情报须经人工环节**筛选与节制**，详见 [SYS-B9002](../10-百科/系统/SYS-B9002-桥与玩家职能.md)。 | 表层 | P1 |
