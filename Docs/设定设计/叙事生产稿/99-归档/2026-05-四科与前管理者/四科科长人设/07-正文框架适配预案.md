# 正文框架适配预案

> **性质**：转正进 `设定设计/设定真值` 前的**框架级改动设计**；本文件**不改正文**，供审定后按条执行。与 [06-玩家情报层级与投放矩阵.md](06-玩家情报层级与投放矩阵.md) 配套。

---

## 第一条　总原则

1. **真值仍归 TIM + SYS-PREV**：年表顺序以 [TIM-04](../../../../设定真值/40-时序与历史/TIM-04-前任管理员执政与第九区事态背景年表.md) 为主本；因果与分层以 [SYS-PREV-BRIDGE](../../../../设定真值/10-百科/系统/SYS-PREV-BRIDGE-前任管理员与第九区废墟事变.md) 为主本；CHR 写**制度锚点人物**，不重复全文年表。  
2. **人物双轨**：`设定真值/10-百科/人物/CHR-*` = 职务、年表锚点、权限、信息分级、互链；`叙事生产稿/` = 印象台词、场景节拍、任务实例（见 [SYS-WORKFORCE](../../../../设定真值/10-百科/规则/SYS-WORKFORCE-员工来源与编组风险.md) 修正案）。  
3. **玩家情报不进真值 frontmatter 的 `spoiler_tier`**：P0–P3 写在 CHR 专节或 [06](06-玩家情报层级与投放矩阵.md)；`spoiler_tier` 仅叙事生产稿。
