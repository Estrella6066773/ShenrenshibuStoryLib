# 街区势力关系简表

## 第一条　表格用途

本表用**一句话定位**概括人事部与主要灰色势力的结构关系，供策划与文案快速对齐口径，避免同一版本中出现彼此矛盾的前提。

## 第二条　关系定位

| 势力 | 与人事部（**人类事务管理部**，[ORG-HRDEPT](../10-百科/组织/赫利俄斯/ORG-HRDEPT-人类事务管理部.md)）的结构关系 |
|------|--------------------------------------|
| “狼群” | 流亡家族组成的联合体，谋求地盘与秩序合法化，较易形成长期外包合作，也可能因责任切割与扶持次序发生变化。 |
| 鬣狗帮 | 对一批机会主义团伙的统称，专在乱局窗口抬价套利，波动大、稳定性低；一旦合作，往往伴随高勒索与反噬风险。 |
| 花豹党 | 强调社区自保与延续，依托地理与人脉门槛，与人事部既有合作空间，也存在摩擦。 |
| 反抗军 | 以改变统治规则为目标，常常是主线推进与道德困境的来源；与人事部可同时存在公开敌对、私下交易与渗透。 |
| 自由人联合阵线 | 跨城武装—政治网络的一支；在本市曾与企业在沦陷区拉锯，主力撤出后仍有残余势力在联合州北部（真值见 [ORG-MOV-FUF](../10-百科/组织/ORG-MOV-FUF-自由人联合阵线.md)）；人事部外宣常与叛乱／废墟叙事捆绑。 |
| 彼岸范式参照运动 | 有组织的社会运动／网络，公开教义美化彼岸一侧的人工智能—社会结盟想象；情报分层上高层可能私下掌握追随对象与前任实例有关的机密，与人事部及舆情接口之间可以是利用、渗透或摩擦（真值见 [ORG-MOV-BRIDGECULT](../10-百科/组织/ORG-MOV-BRIDGECULT-彼岸范式参照运动.md)）。 |
| 人事部 | 制度目标是完成委托并控制损失；原则上可以与各势力做交易，也可能在审计与舆情压力下面临归责。 |
