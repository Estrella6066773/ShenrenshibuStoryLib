# 设定真值

`设定真值/` 目录收录**经审定**的世界观基石、百科条目与叙事合同。此处内容为：

- 创作与策划的**统一真值来源**；  
- 向 `玩家投放/` 切片时的**母本**；  
- 面向大规模玩家考据时的**内部完整版**（对外仍须遵守可见性矩阵）。

## 子目录

| 目录 | 说明 |
|------|------|
| `00-基石/` | 世界纪年、治理秩序、空间模型等**少而硬**的基石条文。 |
| `10-百科/` | 组织、地点、系统、文化概念等百科体条目；以 `id` 为主键。 |
| `20-叙事合同/` | 叙事母题、基调、终局分型与伦理边界；**不属于**剧情细纲。 |

凡新增实体或机制，须登记 `id` 并更新 `治理/id-registry.yaml`。
