# 设定真值

`设定真值/` 目录收录**已定稿**的世界观基石、百科条目与叙事合同。本层级承担下列职能：

- 创作与策划共用的**唯一事实锚点**；  
- 向 `玩家投放/` 切片时的**母本**；  
- 面向深度玩家考据时的**内部完整版**（对外仍要遵守可见性矩阵）。

## 子目录

| 目录 | 说明 |
|------|------|
| `00-基石/` | 世界纪年、治理秩序、空间模型等体量较小、约束较强的条文。 |
| `10-百科/` | 组织、地点、系统、文化概念等百科体条目；以 `id` 为主键。 |
| `20-叙事合同/` | 叙事母题、基调、终局分型与伦理边界；**不是**剧情细纲。 |
| `30-主角与主线/` | 主角（B-9002）处境、主线节拍与矛盾轴的**设计母本**；不写具体台词与关卡脚本（见 [PCM-00](30-主角与主线/PCM-00-使用范围与边界.md)）。 |
| `40-时序与历史/` | 世界历史分期、城市历史母线、关键节点索引与前任管理员背景年表（TIM-00～TIM-04）；与纪年基石 [WLD-00](00-基石/WLD-00-世界纪年与治理总纲.md) 的分工见 [TIM-00](40-时序与历史/TIM-00-时序索引.md)。 |

凡是新增实体或机制，要登记 `id` 并更新 `治理/id-registry.yaml`。
