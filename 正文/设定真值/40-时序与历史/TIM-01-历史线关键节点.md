# 历史线关键节点（TIM-01）

本文将 [TIM-03](TIM-03-世界历史总线.md) 的世界分期与 [TIM-02](TIM-02-城市历史母线.md) 的城市分段汇总为**可按节点检索的对照表**；未定锚点单独列出，以免同一事实在各篇中被改写为彼此矛盾的版本。

## 一、世界—城市对照关系（简表）

| 世界分期（TIM-03） | 与 `[城市A]` 的对应关系（TIM-02） |
|--------------------|-----------------------------------|
| I 联合州主导 | 建城前：殖民扩张与土地占有（城市尚未以现名稳定存在） |
| II 合作社崛起 | 主要在世界线层面展开；对这座城市构成远方的结构与标准竞争压力 |
| III 认知战与禁忌词成型 | 公共话语中「合作社 / 彼岸」标签固化的外部压力 |
| IV 联合州失序与崩解 | 州级能力衰退，为城市被出售提供制度背景 |
| V 后联合州 / 企业辖区化 | 城市被出售予赫利俄斯、企业辖治启动；三群体结构渐成 |
| VI 当下：灰区与叙事拉扯 | 自叙事「当下」往回约 **二十五年**，为前任执政直至废墟固化的重要时段；同一段时间线上的家族战争、「狼群」成形与**自由人联合阵线**战事，见 [TIM-04](TIM-04-前管理者执政与第九区事态背景年表.md)、[ORG-GANG-WOLF](../10-百科/组织/帮派/ORG-GANG-WOLF-狼群.md)、[ORG-MOV-FUF](../10-百科/组织/ORG-MOV-FUF-自由人联合阵线.md) |

## 二、已锚定的相对时间（与百科一致部分）

- **约五十年前（世界线）**：针对合作社的污名化与「彼岸」代称在公共传播中强化（结构层描述，非精确年表）。  
- **自叙事当下回溯二十五年（第九区人事执行与相对年份对齐）**：前任管理者人工智能自执掌第九区并在人事部执行体系中担任关键职务起，至 **现任 B-9002 启用**止，完整相对时间顺序见 **[TIM-04](TIM-04-前管理者执政与第九区事态背景年表.md)**（其中包括：**彼岸西西里岛家族战争**大致落在前任执政约 **第 2 年**；狼群从约 **第 3 年**起持续吸纳外撤家族；**自由人联合阵线**起义、沦陷区拉锯与废墟遭到军事打击约 **第 18 年**等）。  
- **世界线中的家族战争（彼岸西西里岛）**：可与上一条相对年表对照查阅；城市一侧的后果仍以 `[城市A]`／第九区相关条文为准。
