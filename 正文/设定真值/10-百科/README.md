# 百科

本目录以**稳定标识符 `id`** 为主键组织条目，按「组织 / 地点 / 系统 / 文化」四类分文件夹，仅为人类浏览便利；逻辑关系以正文互引与 [治理/id-registry.yaml](../../治理/id-registry.yaml) 为准。世界基石条文见 [设定真值/00-基石/](../00-基石/)；叙事合同条目为 [NAR-00](../20-叙事合同/NAR-00-叙事母题与基调.md)、[NAR-01](../20-叙事合同/NAR-01-主线张力与终局分型.md)、[NAR-02](../20-叙事合同/NAR-02-玩家责任与叙事红线.md)（索引见 [叙事合同 README](../20-叙事合同/README.md)）。

## 新增条目流程

1. 在 [治理/id-registry.yaml](../../治理/id-registry.yaml) 注册新 `id` 与中文标题。  
2. 选用对应子文件夹创建 Markdown，frontmatter 须含 `id`、`title`、`status`、`visibility`。  
3. 若条目涉及对玩家披露边界，须同步标注可见性 tier，并在升版时更新 [玩家投放/](../../玩家投放/README.md) 之 manifest。
