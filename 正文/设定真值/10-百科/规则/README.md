# 制度规则

本目录收录**人类事务管理部**（及集团）在叙事里反复出现的**办事规则与机制**——编组、冲突、优先序、责任切割等。条目 `id` 仍用 `SYS-*` 前缀（历史登记），但**物理位置与「桥」人工智能系统分开**，避免把制度条文误读成 AI 模块。

| 类型 | 说明 |
|------|------|
| **是什么** | 科层制度、默认倾向、玩法约束（可被剧情例外打破，但须因果闭合） |
| **不是什么** | 桥 B-9002、前任管理员等**人工智能实例**（见 [../系统/README.md](../系统/README.md)） |
| **写作时** | 写「人事部规定……」；不要写成「B-9002 系统内置规则」除非剧情明确把制度写进 AI 配置 |
