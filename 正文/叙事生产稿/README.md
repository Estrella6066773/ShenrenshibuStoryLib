# 叙事生产稿（非真值层）

本目录用于存放**任务简报、脚本、关卡叙事、活动文案**等随版本迭代之生产稿。此类文档：

- 允许标注为草案、分支线或已废弃；  
- 与 `设定真值/` 冲突时，**以 `设定真值/` 为准**，除非已获书面特批并登记例外编号；  
- 不得被考据者误认为「设定真值」；对外引用须经 `玩家投放/` 冻结层转写。

## 子目录约定

| 子目录 | 用途 |
|--------|------|
| `简报/` | 任务与活动之策划简报。 |
| `脚本/` | 台本、分支对话、演出说明。 |
| `节拍/` | 可选：节拍表、大纲、结构草图。 |
| `_归档/` | 整包过期内容迁移存放，保留决策痕迹。 |

当前版本尚未将历史剧本自 `.草稿-ai不准修改/` 迁入；迁入时须按版本号分子文件夹，并补充 `canon_refs` 字段说明所依赖之设定 `id`。
