# 治理（正文文档库）

本目录存放**协作规则、命名约定、可见性分级**及 **`id` 注册表**，约束 `正文/` 下增删改与对外投放流程。

| 文件 | 用途 |
|------|------|
| [00-协作与改稿原则.md](00-协作与改稿原则.md) | 真值单一性、层级分工、冲突处理、草稿区边界。 |
| [01-命名文件与ID规则.md](01-命名文件与ID规则.md) | 标识符前缀（含 WLD / ORG / … / TIM）、文件名与 frontmatter。 |
| [02-可见性与剧透分级.md](02-可见性与剧透分级.md) | `visibility`、`spoiler_tier` 与考据披露边界。 |
| [03-内部链接书写规则.md](03-内部链接书写规则.md) | Markdown 相对路径互引、层级速查与断链预防。 |
| [id-registry.yaml](id-registry.yaml) | 已定稿设定条目的 `id`、标题与相对路径（相对 `正文/`）。 |

新增设定实体：**先改** `id-registry.yaml`，**再建** `设定真值/` 内对应 Markdown。管线与导出校验见 [管线/00-导出与一致性检查.md](../管线/00-导出与一致性检查.md)。
