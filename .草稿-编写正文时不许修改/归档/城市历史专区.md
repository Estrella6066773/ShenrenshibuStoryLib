# 城市历史专区（城市A）

本页用于集中讲述`[城市A]`的历史演变，作为时序域的城市级母线文档。

## 使用方式

- 按照"历史时期 -> 结构变化 -> 当下遗留影响"的逻辑组织内容
- 每个历史时期至少需要关联一个可观测后果（舆论变化、制度调整、组织变迁、空间重构）
- 若出现多个版本的叙述，统一在文末记录"版本分歧与证据来源"

## 城市历史总述

`[城市A]`的现代秩序建立在多轮人口与权力重组的基础之上：首先是建城前的殖民扩张与原住人口驱逐阶段，随后殖民者在此建立了城市；接着发生了低成本劳工回流与下城区混居的现象，再到联合州衰落后城市被出售给赫利俄斯并由企业接管，最终形成了身份与资源分配紧密绑定的三群体结构。

## 历史分段

### 1) 殖民扩张与原住人口驱逐期

- 背景：该阶段发生在建城之前，殖民势力首先完成了土地占有与资源控制
- 关键变化：本地土著群体被持续驱逐，殖民者在清空后的空间上建立了`[城市A]`的雏形
- 遗留影响：建城起点即带有不平等的空间分配逻辑，原住群体的历史权利长期处于缺位状态

### 2) 劳工回流与下城区混居期

- 背景：殖民者建立城市并进入扩张期后，需要大量廉价且可替代的劳动力
- 关键变化：被驱逐群体的后代及周边族裔劳工（参考印第安人与墨西哥族裔的现实经验）以劳动人口身份持续流动回城
- 遗留影响：回流劳工与本地贫困人口在下城区混居，形成了长期低保障、高压榨的生存地带

### 3) 联合州衰落与城市出售期

- 背景：联合州的治理能力和财政能力同步衰退
- 关键变化：州政府将`[城市A]`出售给赫利俄斯，城市治理从公权框架切换为企业辖区框架
- 遗留影响：合法性基础从"公民政治参与"转向"运营效率与秩序输出"

### 4) 企业导入与人口重组期

- 背景：赫利俄斯接管后，为稳定生产与管理引入了企业员工体系
- 关键变化：原有居民结构按照治理可控性被重新分层与编码
- 定性设定：十五年前彼岸西西里岛家族战争后，旧式家族秩序被清算，幸存残党外逃至`[城市A]`，成为“狼群”前身。
- 遗留影响：制度通道、配给优先级与治安策略开始按照身份类别进行差异化执行

### 5) 当下结构：三群体并存

- 公司人：赫利俄斯体系导入或认证的正式员工群体，通常拥有更稳定的岗位与资源配给
- 市民：非企业核心编制内的常住居民，维持基础城市运行，资源地位处于中间层次
- 原住民：具有更长本地连续历史、并与回流劳工谱系重叠的群体，多集中于下城区，制度保障相对不足
- 结构后果：三群体在住房、就业、执法暴露和叙事话语权上形成持续张力，构成了城市冲突的主轴

- 每段历史至少需要绑定一个可观测后果（政策差异、空间隔离、风险分层、舆论标签）
